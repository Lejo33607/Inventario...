# Inventario...

Permite registrar un producto con su nombre, precio y cantidad
Luego calcula el costo total del producto y lo muestra en pantalla
Descripción general del programa:
El programa registra el producto dentro del inventario, pidiendole el precio y la cantidad en el proceso. 
Guardando todo y calculando el costo total.
Luego muestra el resultado en la consola.
